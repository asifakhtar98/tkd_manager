About GameCon Tournament Manager

GameCon is a professional tournament management tool built specifically for Taekwondo organizers, gym owners, and martial arts event coordinators. It automates the entire bracket creation process — from athlete registration to seeded brackets with Dojang separation — in under 60 seconds.

Key Features

  • Automatic Bracket Generation: Supports Single Elimination and Double Elimination formats with intelligent seeding.
  • Dojang Separation Algorithm: Automatically seeds players so that students from the same school never face each other in the first round.
  • Live Match Scoring: Touch-friendly scoring interface for ringside volunteers.
  • PDF Tie Sheet Export: Generate professional, gym-floor quality PDF tie sheets.

Tournament Support

  • Supported Formats: Currently supports Single and Double Elimination. Round Robin planned for future updates.
  • Scoring Standards: Follows World Taekwondo (WT) and ITF scoring standards.
  • Cloud-Backed Storage: Secure cloud storage with automatic backups.
  • Scalability: Built for small local competitions to regional championships with 400+ competitors.
